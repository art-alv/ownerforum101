use Astro 

SCOPE:
Design a website/blog for "Hi-there.net"
The top level domain for a website dedicated to providing information and resources for members.

Use a common structure for all the pages as follows:
The website will be divided into pages as follows:
- Home - include About Services and Join.
- Contact
- FAQ
- Blog

Consisting of the following sections:
- Header       Common header for all pages - includes Join.
- SubHeader    Blog-SubHeader
- Cont-Menu    Blog-Menu
- Cont-Content Blog-Content
- Cont-Adds    Blog-Adds
- SubFooter    Blog-SubFooter
- Footer       Common footer for all pages - includes Join.

Dynamic displays:
When entering Hi-there.net
Home Page - will display:

   Header       Common header for all pages - includes Join.
   Sub-Header   Blog-SubHead   post #1 - 
   Cont-Menu    Blog-Menu      post #1 - list of States to select from
   Cont-Content Blog-Content   post #1 -
   Cont-Adds    Blog-Adds      post 
   SubFooter    Blog-SubFooter post #1 -
   Footer       Common footer for all pages - includes Join.

When a state is selected
State Home Page - will display:
   Header       Common header for all pages.
   Sub-Header   Blog-SubHead         post #State
   Cont-Menu    State-Blog-SubHead   post #State-Blog   post #1
   Cont-Content State-Blog-Content   post #State-MenuID post #1
   Cont-Adds    State-Blog-Adds      post 
   SubFooter    Blog-SubFooter       post #State
   Footer       Common footer for all pages.
